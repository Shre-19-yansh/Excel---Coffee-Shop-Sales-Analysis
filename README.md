# ☕ Coffee Sales Dashboard

## 📌 Project Overview
This project explores coffee sales data using Microsoft Excel.
The goal was to better understand behaviour and streamline operations and to analyse sales performance across regions, product type, and time periods, and to build an interactive dashboard for quick insights.

---

## 🔧 Tools Used
- Microsoft Excel,
- Data Cleaning,
- PivotTables,
- Graphs and Charts.

---

## 📂 Dataset
The dataset contains information on coffee sales, including:
- Raw Data:
  Transaction date, time, ID and Quantity
  Store ID and Location
  Product ID, type, category and type
  Unit Price.
- Calculated Data:
  Revenue
  Hour, Day and Month of sales.

## 🔍 Key Insights:
- **June** has the highest sale
- 7am – 10am is the highest sale period
- **Friday** has the most sale
- **Coffee** is the highest selling category.
